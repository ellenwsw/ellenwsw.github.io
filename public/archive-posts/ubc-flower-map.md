# Project Overview
Using tree inventory data from UBC, I created a campus-wide map visualizing the distribution of spring flowering trees across the UBC Vancouver campus. This is the map output from the GIS course taught in the first semester of MGEM program.

## Simplified Process of Map Creation
- Data collection: data was obtained from the PostgreSQL database utilizing SQL commands.
- Data cleaning: extracted the data useful for the purpose of the map, and considered the audience of the map.
- Visualization: in addition to the key map features, map creation also needs to consider map audience and the visual effects.
![UBC spring flower map](/archive-posts/images/ubc-flower-map.png)

# Reflection
- This project combined spatial data processing with cartographic design principles to produce a clear and visually engaging map that communicates seasonal flowering patterns to a broad audience, serving for community flower watching audience. 
- It was an opportunity to apply both my SQL, GIS and cartography skills to a locally meaningful dataset, translating raw inventory records into an accessible and informative spatial product.
- It has to be noted that the color chosen for some of the features are not perfect for visualization effect, which is a key limitation of this output.
